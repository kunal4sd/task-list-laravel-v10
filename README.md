
## Task list laravel 10

Just simple task manager app

<ul>Used Technologies:
    <li>Laravel 10</li>
    <li>PHP 8.2</li>
    <li>MySQL</li>
    <li>Docker</li>
    <li>Alpine.js</li>
    <li>TailwindCSS</li>
</ul>
<ul>Laravel Features:
    <li>Routes</li>
    <li>Models</li>
    <li>Blade Templates</li>
    <li>Pagination</li>
    <li>Factories and Seeder to generate data</li>
    <li>CRUD</li>
    <li>Flash Messages</li>
    <li>Migrations</li>
</ul>
To setup project run in CLI:

Clone Project:

```sh

git clone https://github.com/kunal4sd/task-list-laravel-v10.git

```

Go inside project directory:

```sh

cd task-list-laravel-v10

```

```sh

composer install

```

```sh

cp .env.example .env

```

```sh

php artisan key:generate

```

```sh

php artisan migrate

```

after migration command select yes option to create database schema

run application:

```sh

php artisan serve

```
